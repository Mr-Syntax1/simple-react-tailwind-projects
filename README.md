## 📋 Overview

| Aspect | Description |
|:-------|:------------|
| **🎯 Goal** | Practice React fundamentals through deliberate, small-scope projects |
| **🛠️ Stack** | React (Hooks + Components), Tailwind CSS, Vite |
| **📁 Structure** | One project per folder — isolated, self-contained, and runnable |
| **🧠 Audience** | Front-end developers looking to sharpen React skills |

## 🚀 Quick Start

```# Clone the repository
git clone https://github.com/Mr-Syntax1/simple-react-tailwind-projects.git
```

```# Navigate to project directory
cd simple-react-tailwind-projects
```
```# Install dependencies
npm install
```

```# Start development server
npm run dev
```

## 🎨 Design Philosophy
🧩 Component‑first — every UI piece is reusable and isolated

📱 Mobile‑responsive — Tailwind makes it effortless

♿ Accessible — semantic HTML + ARIA where needed

⚡ Fast — Vite for instant HMR and optimized builds
